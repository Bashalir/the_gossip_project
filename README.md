## The Gossip Project

### Installation

Ruby version 2.5.1
Rails version 5.2.4.3

    git clone git@github.com:Bashalir/the_gossip_project.git
    bundle install
    rails db:migrate
    rails db:seed
    <!-- pour lancer le serveur par default http://localhost:3000 -->
    rails server
    <!-- pour tester l'implementation de la BDD -->
    rails console --sandbox
